Solution :

Time Complexity :


```java
class Solution 
{
    static int minCostClimbingStairs(int[] cost , int N) 
    {
        for (int i = N-3; i >= 0; i--)
            cost[i] += Math.min(cost[i+1], cost[i+2]);
        return Math.min(cost[0], cost[1]);
    }
};
```
